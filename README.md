---

🚀 주요 기능

🔐 인증 시스템

- 버튼 클릭으로 간편 인증
- 역할 자동 부여

👋 입장 / 퇴장 시스템

- 유저 입장 시 환영 메시지
- 퇴장 시 자동 알림

⚠️ 경고 시스템

- 슬래시 명령어로 경고 부여
- 자동 누적 관리
- 🚨 자동 처벌
  - 3회 → 타임아웃
  - 5회 → 킥
  - 7회 → 밴

🎫 티켓 시스템

- 버튼 클릭으로 문의 채널 생성
- 개인 문의 공간 제공
- 버튼으로 티켓 닫기

👑 관리자 패널

- 버튼 UI로 서버 관리
- 공지 작성
- 티켓 전체 삭제
- 간편한 관리 기능

🧹 채팅 관리

- 메시지 일괄 삭제 기능

---

⚙️ 설치 방법

1️⃣ 패키지 설치

pip install -r requirements.txt

2️⃣ 환경변수 설정

- "TOKEN" = 디스코드 봇 토큰

3️⃣ 실행

python main.py
